note is your personal textual scratchpad app for temporariliy storing your ideas, thoughts, links, tasks, grocery lists, phone numbers, email ids and whatever you can think of.

  • Login with any of your OpenId. If you have a google or yahoo account then you already have an OpenId.
  • Save whatever you want in your scratchpad.
  • Data you save is yours. We don't hold any rights to what you save.
  • This is meant to be a temporary store for whatever-you-want-to-save. If something is very important, please keep it saved someplace safe.

HTTP API

A note is uniquely identified by a resource url. A note can be read or updated by isusing a GET or PUT request to this resource url. API responses are in json format.

Resource URL: 

{{ host_url }}/api/note/<secretkey>

Reading

Request: 

GET {{ host_url }}/api/note/<secretkey>

Response: 

{"text": "your note", "last_modified": 1341066212}
where last_modified is unix timestamp.

Example: 

curl -XGET {{ host_url }}/api/note/<secretkey>

Updating

Request: 

PUT {{ host_url }}/api/note/<secretkey>
Form encoded data: text

Response: HTTP status 204.

Example: 

curl -XPUT --data "text=asd" {{ host_url }}/api/note/<secretkey>
